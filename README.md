# IoT Device Integration with AWS Cloud. 

## Descripción General.
Proyecto completo de IoT que implementa un sistema de comunicación bidireccional y seguro entre un dispositivo ESP32 y la plataforma cloud AWS IoT Core. El sistema demuestra las mejores prácticas de desarrollo embebido, seguridad por certificados, procesamiento serverless y integración de servicios en la nube.

## Objetivos del Proyecto.
Implementar una conexión segura MQTT entre ESP32 y AWS IoT Core.
